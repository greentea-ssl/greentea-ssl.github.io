RoboCup JapanOpen 2025 SSL参戦！

  • 2025/05/12
  • PR, event

我々GreenTeaは今年もRoboCup JapanOpen(5/2 - 5)に参加しました．開催場所は去年同様滋賀ダイハツアリーナです．車でのアクセスはよくて便利ですね．来年もここがいいです．

今年はタイからの学生チームOrcaBOTとドイツからのチームTIGERs Mannheim (通称TIGERs)が参戦していました．TIGERsは非常に有名なチームで彼らのロボットをじかに見ることができる非常に貴重な機会でした．
試合参加はこれで四度目ですが残念ながら予選リーグ敗退となってしまいました．一方でロボットからの状態をパドックでリモートに確認するシステムが動いたり，ローカルカメラシステムがBall placementでうまく機能してることが確認できたりしたのでその点は満足です．



試合後はOrcaBOTと情報交換をしたり，TIGERsのチームにお邪魔して間近でロボットを見せてもらっていろいろ解説してもらったりしました．TIGERsには何か強さの特別な秘訣があるに違いない！と意気込んで乗り込みましたが実際のところ基本的なアイディアは私たちとあまり違いはなく，単に圧倒的な完成度の差を見せつけられる形となってしまいました……．来年は機体もAIも一新するので今度こそ上位入賞したいところです．




例年通り試合外でもいろいろやっていました．まずは例年通り配信をやっていました．今年はまともな出力のスピーカー(私物)で会場にも音を流すようにしたので特に最終日は去年より観衆が多かった気がします．あとStatus BoardやVisionをインターネットから見れるようにしました．これは好評でした．

今年も多くの学びと刺激を得ることができた大会でした．応援してくださった皆様，そして何よりも大会をつつがなく進行してくださった運営の方々，本当にありがとうございました．
GreenTeaの次回作にご期待ください！！！
