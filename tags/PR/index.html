гғӯгғңгғғгғҲгҒ«гӮ«гғЎгғ©д№—гҒӣгҒҰгҒҝгҒҹ

  • 2023/12/24
  • PR, AI, SSL-Vision

гҒ“гҒ®иЁҳдәӢгҒҜпјҢssl-zemi 12жңҲ(2023/12/17)гҒ§зҷәиЎЁгҒ—гҒҹеҶ…е®№гӮ’з·ЁйӣҶгҒ—гҒҹзү©гҒ§гҒҷпјҺ

SSL-visionгҒ®жҰӮиҰҒгҒЁиӘІйЎҢ

SSLгҒҜеӨ©дә•гҒ«гҒӨгҒ‘гӮүгӮҢгҒҹгӮ«гғЎгғ©гҒ®жғ…е ұгӮ’гӮӮгҒЁгҒ«пјҢеҗ„гғҒгғјгғ гҒҜгғӯгғңгғғгғҲгҒёе‘Ҫд»ӨгӮ’йҖҒгӮҠгҒҫгҒҷпјҺеӨ©дә•гҒӢгӮүгҒ®иҰ–зӮ№гҒ§гҒӮгӮӢгҒ®гҒ§пјҢгғӯгғңгғғгғҲгҒ®дҪҚзҪ®гӮ„гғңгғјгғ«гҒ®дҪҚзҪ®гӮ’жӨңеҮәгҒҷгӮӢгҒ®гҒҜз°ЎеҚҳгҒ§гҒҷпјҺ

гҒ—гҒӢгҒ—пјҢеӨ©дә•гҒ®й«ҳгҒ•гӮ„гӮ«гғЎгғ©гҒ®й…ҚзҪ®гҒ«гӮҲгҒЈгҒҰгҒҜпјҢгғӯгғңгғғгғҲгҒ«гӮҲгҒЈгҒҰгғңгғјгғ«гҒҢгҒҝгҒҲгҒӘгҒҸгҒӘгҒЈгҒҹгӮҠпјҢз…§жҳҺзҠ¶жіҒгҒ«гӮҲгӮҠгғӯгғңгғғгғҲгӮ„гғңгғјгғ«гҒ®жӨңеҮәгҒҢе®үе®ҡгҒ—гҒӘгҒӢгҒЈгҒҹгӮҠгҒҷгӮӢгҒ“гҒЁгҒҢгҒӮгӮҠгҒҫгҒҷпјҺ

гҒҫгҒҹпјҢгғҚгғғгғҲгғҜгғјгӮҜгӮ’йҖҡгҒЈгҒҰпјҢеҗ„гғҒгғјгғ гҒ§еҲ©з”ЁгҒ§гҒҚгӮӢгҒҫгҒ§гҒ«пјҢгғ¬гӮӨгғҶгғігӮ·гҒҢ100 ms(60 fpsпјҢ6F)зЁӢеәҰеӯҳеңЁгҒ—гҒҫгҒҷпјҺгҒқгҒ“гҒ§пјҢSSL-visonгӮ·гӮ№гғҶгғ гҒ®жҖ§иғҪгӮ„дјҡе ҙгҒ«гӮҲгӮӢе·®гҒ«гӮҲгӮӢеҪұйҹҝгӮ’дҪҺжёӣгҒҷгӮӢгҒҹгӮҒгҒ«пјҢгғӯгғңгғғгғҲгҒ«гӮ«гғЎгғ©гӮ’ијүгҒӣгҒҰгҒҝгҒҫгҒ—гҒҹпјҺ

ијүгҒӣгҒҰгҒҝгҒҹ

гғӯгғңгғғгғҲгҒ«Raspberry piгҒЁгӮ«гғЎгғ©гӮ’гҒқгҒ®гҒҫгҒҫијүгҒӣгҒҰгҒ—гҒҫгҒ„гҒҫгҒҷпјҺпјҲгғўгғҗгӮӨгғ«гғҗгғғгғҶгғӘгҒҜй–ӢзҷәдёӯгҒ®гӮӮгҒ®гҒ§пјҢе®ҹйҡӣгҒ«гҒҜеҸ–гӮҠеӨ–гҒ—гҒҫгҒҷпјҺпјү

гӮ«гғЎгғ©(OV5647)гҒӢгӮүYUVеҪўејҸпјҢ640*480пјҢ90 fpsгҒ§з”»еғҸгӮ’еҸ–еҫ—гҒ—гҒҫгҒҷпјҺRaspberry pi 4гҒ«гӮҲгӮҠз”»еғҸеҮҰзҗҶиЎҢгҒ„пјҢжӨңеҮәгҒ—гҒҹгғңгғјгғ«дҪҚзҪ®пјҲгғӯгғңгғғгғҲеә§жЁҷзі»пјүгӮ’гғӯгғңгғғгғҲгӮ’еҲ¶еҫЎгҒҷгӮӢгғһгӮӨгӮігғі(STM32H7)гҒ«UARTйҖҒдҝЎгҒ—гҒҫгҒҷпјҺ
Raspberry pi гӮ’з”»еғҸеҮҰзҗҶгғўгӮёгғҘгғјгғ«гҒЁгҒ—гҒҰз–ҺгҒӘзөҗеҗҲгҒЁгҒҷгӮӢгҒ“гҒЁгҒ§пјҢж—ўеӯҳгҒ®гғӯгғңгғғгғҲгӮ·гӮ№гғҶгғ гҒёгҒ®еҪұйҹҝгӮ’е°‘гҒӘгҒҸпјҢгӮ«гғЎгғ©гӮ·гӮ№гғҶгғ гҒ®ијүгҒЈгҒҰгҒ„гҒӘгҒ„ж©ҹдҪ“гҒЁгӮӮе…ұеӯҳгҒ•гҒӣгӮ„гҒҷгҒ„ж§ӢжҲҗгҒЁгҒ—гҒҰгҒ„гҒҫгҒҷпјҺд»ҠеҫҢгҒ®гӮ·гӮ№гғҶгғ гҒ®й–ӢзҷәзҠ¶жіҒпјҢзЁјеғҚжҷӮгҒ®е®үе®ҡжҖ§гҒӘгҒ©гҒ«гӮҲгҒЈгҒҰгҒҜпјҢзөҗеҗҲгҒ®зЁӢеәҰгҒҜеӨүгӮҸгӮӢеҸҜиғҪжҖ§гҒҢгҒӮгӮҠгҒҫгҒҷпјҺ

гғңгғјгғ«жӨңеҮә

YUVгҒ®дҝЎеҸ·гҒҜпјҢијқеәҰдҝЎеҸ·гҒЁ2гҒӨгҒ®иүІе·®дҝЎеҸ·гҒӢгӮүж§ӢжҲҗгҒ•гӮҢгҒҰгҒ„гҒҫгҒҷпјҺгҒ“гҒ®иүІе·®дҝЎеҸ·2гҒӨгҒ®е·®гӮ’еҸ–гӮӢгҒ“гҒЁгҒ§пјҢгӮӘгғ¬гғігӮёгҒЈгҒҪгҒ•гҒ«гӮҲгӮҠжҳҺжҡ—гҒҢеӨүеҢ–гҒҷгӮӢгӮ°гғ¬гғјгӮ№гӮұгғјгғ«з”»еғҸгӮ’еҸ–еҫ—гҒҷгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷпјҺ
гҒ“гҒ®з”»еғҸгҒ«еҜҫгҒ—гҒҰпјҢгҒ„гҒҸгҒӨгҒӢиЁӯе®ҡгҒ—гҒҹй«ҳгҒ•гҒ§пјҢжЁӘж–№еҗ‘гҒ«иө°жҹ»гҒ—гҒӘгҒҢгӮүгӮЁгғғгӮёгӮ’жӨңеҮәгҒ—гҒҫгҒҷпјҺжӨңеҮәгҒ•гӮҢгҒҹгӮЁгғғгӮёгҒ®дёӯзӮ№гӮ’зөҗгӮ“гҒ з·ҡдёҠгӮ’ж”№гӮҒгҒҰпјҢзёҰж–№еҗ‘гҒ«иө°жҹ»гҒ—гҒӘгҒҢгӮүгӮЁгғғгӮёгӮ’жӨңеҮәгҒ—пјҢгғңгғјгғ«гҒ®дёӯеҝғеә§жЁҷгӮ’зү№е®ҡгҒ—гҒҫгҒҷпјҺгҒ“гҒ®жүӢжі•гҒҜTIGERs MANNHEIMгҒ®жүӢжі•гӮ’еҸӮиҖғгҒ«гҒ—гҒҰгҒ„гҒҫгҒҷпјҺ

з–ҺгҒ«жҺўзҙўгҒҷгӮӢгҒ“гҒЁгҒ«гӮҲгӮҠпјҢй«ҳйҖҹеҢ–гӮ’гҒ—гҒҰгҒ„гҒҫгҒҷпјҺгҒҫгҒҹпјҢйқһеёёгҒ«дёҰеҲ—жҖ§гӮ’дёҠгҒ’гӮ„гҒҷгҒ„жүӢжі•гҒ§гҒҷпјҺ

еә§жЁҷеӨүжҸӣ

гӮ№гӮҜгғӘгғјгғіеә§жЁҷзі»гҒ®гғңгғјгғ«дҪҚзҪ®гҒӢгӮүпјҢгғӯгғңгғғгғҲеә§жЁҷзі»гҒ«еӨүжҸӣгҒ—гҒҫгҒҷпјҺ

гғӯгғңгғғгғҲеә§жЁҷзі»->гӮ№гӮҜгғӘгғјгғіеә§жЁҷзі»

гӮ№гӮҜгғӘгғјгғіеә§жЁҷзі»гҒӢгӮүгғӯгғңгғғгғҲеә§жЁҷзі»гҒёгҒ®йҖҶеӨүжҸӣгӮ’иҖғгҒҲгӮӢеүҚгҒ«пјҢгғӯгғңгғғгғҲеә§жЁҷзі»гҒӢгӮүгӮ№гӮҜгғӘгғјгғіеә§жЁҷгҒёгҒ®й ҶеӨүжҸӣгӮ’иҖғгҒҲгҒҫгҒҷпјҺ

гҒ“гӮҢгҒҜпјҢгғӯгғңгғғгғҲеә§жЁҷзі»->гӮ«гғЎгғ©еә§жЁҷзі»гҒЁпјҢгӮ«гғЎгғ©еә§жЁҷзі»->гӮ№гӮҜгғӘгғјгғіеә§жЁҷзі»гҒёгҒ®еӨүжҸӣгҒ«еҲҶеүІгҒ§гҒҚгҒҫгҒҷпјҺ
гғӯгғңгғғгғҲеә§жЁҷзі»->гӮ«гғЎгғ©еә§жЁҷзі»гҒ®еӨүжҸӣгҒҜиҰ–йҮҺеӨүжҸӣгҒЁе‘јгҒ°гӮҢгӮӢж“ҚдҪңгҒ§иЎҢгҒ„гҒҫгҒҷпјҺе…·дҪ“зҡ„гҒ«гҒҜ

  1. гғҷгӮҜгғҲгғ«гӮ’и¶ігҒ—гҒҰгӮ«гғЎгғ©гҒ®дҪҚзҪ®гҒҫгҒ§еҺҹзӮ№гӮ’еӢ•гҒӢгҒҷ
  2. еӣһи»ўиЎҢеҲ—гӮ’дҪҝгҒЈгҒҰгӮ«гғЎгғ©гҒ®еҗ‘гҒҚгҒ«еә§жЁҷгӮ’еҗҲгӮҸгҒӣгӮӢ

гҒ“гҒЁгҒ§е®ҹзҸҫгҒ§гҒҚгҒҫгҒҷпјҺ

гӮ«гғЎгғ©еә§жЁҷзі»->гӮ№гӮҜгғӘгғјгғіеә§жЁҷгҒҜйҖҸиҰ–еӨүжҸӣгҒЁе‘јгҒ°гӮҢгӮӢж“ҚдҪңгӮ’иЎҢгҒЈгҒҹгҒӮгҒЁгҒ«еҺҹзӮ№гӮ’з”»йқўе·ҰдёҠгҒ«гҒҡгӮүгҒҷгҒ“гҒЁгҒ§е®ҹзҸҫгҒ§гҒҚгҒҫгҒҷпјҺOpenCVгҒ®е ҙеҗҲпјҢгҒ“гҒ®еӨүжҸӣгҒҜжҺЁе®ҡгҒ•гӮҢгҒҹгӮ«гғЎгғ©иЎҢеҲ—гӮ’дҪҝгҒҶгҒ“гҒЁгҒ§е®ҹзҸҫгҒ•гӮҢгҒҫгҒҷпјҺгҒҸгӮҸгҒ—гҒҸгҒҜгӮ«гғЎгғ©гӮӯгғЈгғӘгғ–гғ¬гғјгӮ·гғ§гғігҒЁ3ж¬Ўе…ғеҶҚж§ӢжҲҗгӮ’еҸӮз…§гҒ—гҒҰгҒҸгҒ гҒ•гҒ„пјҺ

гӮ№гӮҜгғӘгғјгғіеә§жЁҷзі»->гҖҖгғӯгғңгғғгғҲеә§жЁҷзі»

з¶ҡгҒ„гҒҰйҖҶеӨүжҸӣгӮ’иҖғгҒҲгҒҫгҒҷпјҺй ҶеӨүжҸӣгҒ§е°ҺеҮәгҒ—гҒҹејҸгӮ’йҖҶгҒ«и§ЈгҒ‘гҒ°гҒ„гҒ„гҒ®гҒ§гҒҷгҒҢпјҢ2ж¬Ўе…ғгҒӢгӮү3ж¬Ўе…ғгҒёгҒ®еӨүжҸӣгҒӘгҒ®гҒ§дёҚиүҜиЁӯе®ҡе•ҸйЎҢгҒЁгҒӘгӮҠгҒҫгҒҷпјҺ
гҒқгҒ®гҒҹгӮҒгҖҢгғңгғјгғ«гҒ®zеә§жЁҷгҒҜ0гҒ§гҒӮгӮӢгҖҚгҒЁгҒ„гҒҶеҲ¶зҙ„гӮ’гҒӢгҒ‘гҒҫгҒҷпјҺ
гӮ№гӮҜгғӘгғјгғіеә§жЁҷгҒӢгӮүгӮ«гғЎгғ©еә§жЁҷгҒёгҒ®еӨүжҸӣгҒҜдёӢгҒ®гӮҲгҒҶгҒ«гҒӘгӮҠгҒҫгҒҷпјҺгӮ«гғЎгғ©еә§жЁҷгҒ®zеә§жЁҷгҒҜйҖҡеёёеҘҘиЎҢгҒҚгҒЁзӯүгҒ—гҒ„гҒ§гҒҷгҒҢпјҢд»ҠеӣһгҒҜжұәгҒҫгӮүгҒӘгҒ„гҒ®гҒ§1гҒЁгҒ—гҒҰиЁҲз®—гҒ—гҒҫгҒҷпјҺOpenCVгҒ®е ҙеҗҲпјҢгӮ«гғЎгғ©иЎҢеҲ—гҒ®йҖҶиЎҢеҲ—гӮ’жҺЁе®ҡгҒ—гҒҹгӮ№гӮҜгғӘгғјгғіеә§жЁҷгҒ«гҒӢгҒ‘гҒҰжұӮгӮҒгҒҫгҒҷпјҺеҝ…иҰҒгҒӘгӮүundistortPointsгӮ’е…ҲгҒ«йҒ©з”ЁгҒ—гҒҫгҒҷпјҺ

гӮ«гғЎгғ©еә§жЁҷзі»гҒӢгӮүгғҜгғјгғ«гғүеә§жЁҷзі»гҒёгҒ®еӨүжҸӣгҒҜд»ҘдёӢгҒ§гҒҷпјҺгҒ“гҒ®жҷӮгҒ«гғӯгғңгғғгғҲеә§жЁҷгҒ®zгҒҢ0гҒ«гҒӘгӮӢгӮҲгҒҶгҒ«гӮ№гӮұгғјгғ«еӨүж•°sгӮ’жұәгӮҒгҒҫгҒҷпјҺ

зөҗжһң


гҒ“гҒЎгӮүгҒҢгғңгғјгғ«дҪҚзҪ®жӨңеҮәгҒ®ж§ҳеӯҗгҒ§гҒҷпјҺгғўгғҺгӮҜгғӯз”»еғҸгҒҢгӮЁгғғгӮёеҮҰзҗҶгҒ«дҪҝгӮҸгӮҢгҒҰгҒ„гӮӢз”»еғҸгҒ§пјҢгӮ«гғ©гғјз”»еғҸгҒ®й»’з·ҡгҒҜгӮЁгғғгӮёеҮҰзҗҶгҒ«дҪҝгҒҶиЎҢгӮ’зӨәгҒ—гҒҰгҒ„гҒҫгҒҷпјҺгӮ«гғ©гғјз”»еғҸгҒ«жҸҸз”»гҒ•гӮҢгҒҰгҒ„гӮӢз·‘зӮ№гҒҢжҺЁе®ҡгҒ•гӮҢгҒҹгғңгғјгғ«дёӯеҝғгҒ§гҒҷпјҺдёӯеҝғдҪҚзҪ®гҒ®x,y,zеә§жЁҷгҒҢcmеҚҳдҪҚгҒ§гӮ«гғ©гғјз”»еғҸе·ҰдёӢгҒ«иЎЁзӨәгҒ•гӮҢгҒҰгҒ„гҒҫгҒҷпјҺеҸідёӢгҒ«еҮәгҒҰгӮӢгҒ®гҒҢгғ•гғ¬гғјгғ еҪ“гҒҹгӮҠгҒ®еҮҰзҗҶжҷӮй–“гҒ§гҒҷпјҺGUIгӮ’иЎЁзӨәгҒ—гҒҰгҒ„гӮӢгҒӣгҒ„гӮӮгҒӮгӮҠгҒҫгҒҷгҒҢгҒқгӮҢгҒ»гҒ©йҖҹгҒҸгҒӘгҒ„гҒ§гҒҷгҒӯпјҺ
